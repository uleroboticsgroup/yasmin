ROS Serialization Demo (Python)

This tutorial demonstrates how to receive and deserialize ROS interfaces that were serialized by a C++ state. The PoseReaderState is a Python state that reads serialized geometry_msgs/Pose bytes from the blackboard and deserializes them using rclpy.serialization. This enables cross-language data exchange within the same state machine.

Python State: PoseReaderState

The PoseReaderState reads pose_bytes from the blackboard (written by a C++ PoseWriterState), deserializes it into a geometry_msgs.msg.Pose, and logs the position and orientation values. 

from geometry_msgs.msg import Pose
from rclpy.serialization import deserialize_message

import yasmin
from yasmin import Blackboard, State


class PoseReaderState(State):
    """
    Represents a state that reads and deserializes a Pose message
    from the blackboard.
    """

    def __init__(self) -> None:
        """
        Initializes the PoseReaderState instance, setting up the outcome.
        """
        super().__init__(outcomes=["outcome2"])
        self.set_description(
            "Reads a serialized Pose message from the blackboard, "
            "deserializes it, and logs the position and orientation."
        )
        self.add_input_key(
            "pose_bytes",
            "Serialized Pose message stored as bytes in the blackboard.",
        )
        self.add_input_key(
            "pose_bytes__type",
            "Type information for the serialized Pose message.",
            "geometry_msgs/msg/Pose",
        )

    def execute(self, blackboard: Blackboard) -> str:
        """
        Executes the logic for the PoseReaderState.
        """
        yasmin.YASMIN_LOG_INFO("Executing state POSE_READER")

        pose_type = blackboard["pose_bytes__type"]
        pose_bytes = blackboard["pose_bytes"]
        pose = deserialize_message(pose_bytes, Pose)

        yasmin.YASMIN_LOG_INFO(f"Stored type: {pose_type}")
        yasmin.YASMIN_LOG_INFO(
            f"Position: x={pose.position.x}, "
            f"y={pose.position.y}, z={pose.position.z}"
        )
        yasmin.YASMIN_LOG_INFO(
            f"Orientation: x={pose.orientation.x}, "
            f"y={pose.orientation.y}, z={pose.orientation.z}, "
            f"w={pose.orientation.w}"
        )

        return "outcome2"

XML State Machine

This state is used together with the C++ PoseWriterState via the demo_3.xml state machine definition (loaded by the YasminFactory). See the C++ ROS Serialization tutorial for the full XML definition and factory usage.
