API Reference

This page documents the public API of the core yasmin package. It covers all classes needed to create states, state machines, and concurrent containers in both C++ and Python.

State

The base class for all states. To create a custom state, subclass State and implement the execute() method. The constructor receives the set of valid outcome strings that execute() may return.

Constructor

C++

State(const std::set<std::string> &outcomes);

Python

State(outcomes: list[str])

Example

C++

#include "yasmin/state.hpp"
#include "yasmin/blackboard.hpp"

class FooState : public yasmin::State {
public:
    FooState() : yasmin::State({"outcome1", "outcome2"}) {
        this->set_description("Performs the Foo operation.");
        this->add_output_key("foo_str", "Result string produced by FooState.");
    }

    std::string execute(yasmin::Blackboard::SharedPtr blackboard) override {
        blackboard->set<std::string>("foo_str", "Hello from Foo");
        return "outcome1";
    }
};

Python

from yasmin import State, Blackboard

class FooState(State):
    def __init__(self):
        super().__init__(["outcome1", "outcome2"])
        self.set_description("Performs the Foo operation.")
        self.add_output_key("foo_str", "Result string produced by FooState.")

    def execute(self, blackboard: Blackboard) -> str:
        blackboard["foo_str"] = "Hello from Foo"
        return "outcome1"

StateMachine

StateMachine inherits all State metadata methods (set_description, add_input_key, add_output_key, declare_parameter, etc.).

Example

C++

#include "yasmin/state_machine.hpp"
#include "yasmin/blackboard.hpp"

auto sm = std::make_shared<yasmin::StateMachine>(
    std::set<std::string>{"outcome1", "outcome2"});

sm->set_description("A simple state machine.");
sm->add_input_key("foo_str", "Input string for FooState.");

sm->add_state("FOO", std::make_shared<FooState>(),
              {{"outcome1", "BAR"}, {"outcome2", "outcome2"}});
sm->add_state("BAR", std::make_shared<BarState>(),
              {{"outcome3", "outcome1"}});

auto blackboard = std::make_shared<yasmin::Blackboard>();
blackboard->set<std::string>("foo_str", "hello");
std::string outcome = (*sm)(blackboard);

Python

from yasmin import StateMachine, Blackboard

sm = StateMachine(outcomes=["outcome1", "outcome2"])
sm.set_description("A simple state machine.")
sm.add_input_key("foo_str", "Input string for FooState.")

sm.add_state("FOO", FooState(),
             transitions={"outcome1": "BAR", "outcome2": "outcome2"})
sm.add_state("BAR", BarState(),
             transitions={"outcome3": "outcome1"})

blackboard = Blackboard()
blackboard["foo_str"] = "hello"
outcome = sm(blackboard)

CbState

A convenience state that wraps a plain callback function. Use it when the state logic is too simple to justify a dedicated subclass.

Constructor

C++

CbState(const std::set<std::string> &outcomes,
        std::function<std::string(Blackboard::SharedPtr)> callback);

Python

CbState(outcomes: list[str], callback: Callable[[Blackboard], str])

The callback receives the shared Blackboard and must return one of the declared outcome strings. CbState inherits all State metadata methods.

Example

C++

#include "yasmin/cb_state.hpp"

auto greeting_state = std::make_shared<yasmin::CbState>(
    std::set<std::string>{"done"},
    [](yasmin::Blackboard::SharedPtr bb) -> std::string {
        YASMIN_LOG_INFO("Hello from CbState");
        return "done";
    });

greeting_state->set_description("Prints a greeting message.");
sm->add_state("GREET", greeting_state, {{"done", "outcome1"}});

Python

from yasmin import CbState, Blackboard
import yasmin

def greeting_cb(blackboard: Blackboard) -> str:
    yasmin.YASMIN_LOG_INFO("Hello from CbState")
    return "done"

greeting_state = CbState(outcomes=["done"], callback=greeting_cb)
greeting_state.set_description("Prints a greeting message.")
sm.add_state("GREET", greeting_state, transitions={"done": "outcome1"})

Concurrence

Concurrence extends State and executes multiple child states in parallel threads. When all threads finish, the combined outcomes are matched against an outcome map to determine the overall outcome. If no rule matches, the default outcome is returned.

Constructor

C++

Concurrence(
    const StateMap &states,
    const std::string &default_outcome,
    const OutcomeMap &outcome_map,
    const ParameterMappingsMap &parameter_mappings = {});

Python

Concurrence(
    states: dict[str, State],
    default_outcome: str,
    outcome_map: dict[str, dict[str, str]])

states maps logical names to State objects. outcome_map maps desired overall outcome strings to dictionaries of required per-state outcomes, e.g.: 

outcome_map = {
    "overall_success": {
        "STATE_A": "succeeded",
        "STATE_B": "done",
    }
}

This rule is satisfied when STATE_A returns "succeeded" and STATE_B returns "done". Rules are evaluated after all child threads finish. The first matching rule in iteration order is used.

Example

C++

#include "yasmin/concurrence.hpp"

auto foo_state = std::make_shared<FooState>();
auto bar_state = std::make_shared<BarState>();

auto concurrence_state = std::make_shared<yasmin::Concurrence>(
    yasmin::StateMap{{"FOO", foo_state}, {"BAR", bar_state}},
    "defaulted",
    yasmin::OutcomeMap{
        {"outcome1", {{"FOO", "outcome1"}, {"BAR", "outcome3"}}},
        {"outcome2", {{"FOO", "outcome2"}, {"BAR", "outcome3"}}},
    });

concurrence_state->set_description("Runs FOO and BAR in parallel.");

sm->add_state("CONCURRENT", concurrence_state,
              {{"outcome1", "outcome1"},
               {"outcome2", "outcome2"},
               {"defaulted", "outcome2"}});

Python

from yasmin import Concurrence

foo_state = FooState()
bar_state = BarState()

concurrence_state = Concurrence(
    states={"FOO": foo_state, "BAR": bar_state},
    default_outcome="defaulted",
    outcome_map={
        "outcome1": {"FOO": "outcome1", "BAR": "outcome3"},
        "outcome2": {"FOO": "outcome2", "BAR": "outcome3"},
    },
)
concurrence_state.set_description("Runs FOO and BAR in parallel.")

sm.add_state(
    "CONCURRENT",
    concurrence_state,
    transitions={
        "outcome1": "outcome1",
        "outcome2": "outcome2",
        "defaulted": "outcome2",
    },
)
